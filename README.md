## 🌟 Features
- **User Authentication** - JWT-based login/register
- **Task Management** - Create, read, update, and delete tasks
- **Project Organization** - Organize tasks into projects with progress tracking
- **Priority Levels** - Low, Medium, High priority with colour coding
- **Status Tracking** - To Do, In Progress, Done
- **Dashboard** - Visual overview with statistics
- **Lazy Loading** - All pages loaded on demand (React.lazy + Suspense)
- **Responsive Design** - Works on desktop and mobile

## 🛠️ Tech Stack
### Frontend (Deployed on Vercel)
- **React 18** - UI library
- **TypeScript** - Type-safe JavaScript
- **Redux Toolkit** - State management
- **Tailwind CSS** - Styling
- **React Router** - Navigation
- **Vite** - Build tool

### Backend (ASP.NET Core 8.0)
- **ASP.NET Core 8.0** - Web API framework
- **Entity Framework Core** - ORM
- **SQL Server** - Database
- **JWT** - Authentication
